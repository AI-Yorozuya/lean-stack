# 劇本・一頁式品牌官網（speedrun v1）

> 選了此劇本後，教練照本檔演。目標：**最少對話 → 一版真的能收詢問的一頁網站**。
> 起手種子：`scenario/landing` 分支（公開頁骨架 + 詢問表單後端）。分支尚未切出前，
> 用現行 code 也能演——把電商前台當成待改造的空白頁，聯絡表單先接 `inquiry` 端點。

## v1 的誠實 MVP（一定端到端、不做假頁）

一頁公開網站，**訪客送出詢問 → 真的存進後端 → 後台看得到**：
- **前台（:5175）**：Hero（招牌＋標語＋一顆 CTA）→ 亮點卡 ×3 → 關於 → 聯絡表單。
- **送出**：`POST /web/inquiry`（存 `Inquiry`：姓名/聯絡方式/訊息/來源時間）。
- **後台（:5174）**：「詢問」列表——老闆看得到誰來問過。
- 這就是全端：他只填了內容，客人送的詢問真的落進他的後台。

## 對話（AskUserQuestion，四問，每問都給預設值）

**Q1 招牌在做什麼**（決定範例文案與語氣）
options：接案／個人品牌 · 工作室／團隊 · 專業服務（顧問/設計/律師）· 店家／餐飲 · 其他
→ Other 就讓他一句話講，map 到最近的一類。

**Q2 一句話說你在做什麼**（Hero 標語）
用 AskUserQuestion 的 Other 或直接問；沒靈感就用「{名字}－{Q1類別}」先擺，之後能改。

**Q3 選個氣質**（一次定主題 token，別逐色問）
options：
- **貼個我喜歡的網站當參考**（AI-native 升級路，見下）
- 極簡黑白——大留白、無襯線、黑白灰＋一個重點色
- 溫暖手作——米色紙感、襯線標題、磚紅點綴
- 專業沉穩——深藍/石墨、俐落、商務感
- 明亮活潑——高飽和、圓角、年輕
→ 選預設氣質：寫進 `lean-web` 主題 token（`src/style.css` 的 `:root`），只改這裡就換膚。

**Q3-參考路**（選了「貼網站參考」才走）：
- 請他貼一個喜歡的網站網址（可多個：「這個的排版＋那個的顏色」）。
- 教練 `WebFetch` 讀它，**只抽抽象特質**：配色/字型氣質、版型節奏（幾段、hero 樣式、要不要大圖）、語氣。
- **鐵則：參考不複製。** 抽出的特質 → 主題 token＋版型；文案一律用他 Q1/Q2 的資訊**重新生成**。
  - 絕不搬：對方的 logo、圖片素材、文案原句、品牌名。
  - 跟他講白：「我照它的**氣質**做你的頁，不是複製它——內容都是你的。」
- 讀不到/太複雜 → 退回四個預設氣質，不卡關。

**Q4 來的人你最想要他做什麼**（決定 CTA 文字＋表單欄位）
options：
- 留聯絡方式問我 → CTA「聯絡我」，表單＝姓名＋Email/電話＋訊息
- 預約免費諮詢 → CTA「預約諮詢」，表單多一個「想聊的時段」
- 加 LINE／傳訊 → CTA「加我 LINE」，表單＝暱稱＋訊息（＋放 LINE ID）
- 看作品再說 → CTA「看看我的作品」，先錨到亮點區、表單放頁尾

## 教練生成（四答之後，一氣做完再報告）

1. 把 `lean-web` 收斂成單頁（Hero／亮點／關於／表單四段），套 Q3 的主題 token、填 Q1/Q2 的文案。
2. 後端：`apps/inquiry`（model `Inquiry` 繼承 TimeStampedModel、`apis.py` 一個 Router、`schemas.py`）→ `core/api.py` 註冊 → 容器內 migrate。
3. 前台表單接 `POST /web/inquiry`；後台加「詢問」列表頁（用 `DataTable`）。
4. 開兩個網址給他看：「這是你的一頁——填一筆詢問試試，回後台就看得到。」
5. 存檔：「這版會動了，幫你存了個進度點『官網 v1』。」

> 別一次問完再做全部靜默——**每答一題可即時反映的就先反映**（改標題、換色），讓他邊選邊看到頁在長。

## Boss「亮相」（此劇本的通關）

把公開網址分享出去 / 收到**第一則真詢問**（不是自己測的那筆）。
台詞：「這頁現在活在網路上了。傳給一個人看看——等第一則不是你自己填的詢問進來，你就亮相成功了。」
（真上線＝部署，屬 T5；v1 先本機亮相／內網分享，敢交付留大魔王。）

## 存檔節奏（取代砸店）

不製造假意外。存檔在**他真的要改、怕改壞**的自然時機：
- 表單接通、頁面會動 → 主動存「官網 v1」。
- 他說「我想大改版面」→ 動手前先存還原點。
- 一段落穩了 → 提議備份上雲（選配支線「辦個家」，見 SKILL.md 0.3）。
