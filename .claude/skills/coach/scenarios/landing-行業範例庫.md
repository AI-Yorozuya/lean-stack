# landing・行業範例庫（複利資產：每遇一個新行業，就往這裡存一組）

> 用法鐵律：**公式化參考，個人化生成**——這裡給的是公式與骨架，教練必須融合玩家的實際資訊（店名、地點、特色）生成他專屬的版本。
> **絕不逐字照搬**：兩個甜點店不能拿到同一句標語（從公式生變體），否則「專屬」就是假的。
> 沉澱規則：玩家的行業不在庫裡→照通用公式現場生成，**做完把該行業的骨架補進本檔**（一次遊玩，永久資產）。

## 標語通用公式（Q1 各行業都適用，擇一套入）

1. **具體物×情感**：「{招牌動作/產品}，{給誰}{什麼感受}」——例：一顆一顆手作的常溫甜點，把日子過甜一點。
2. **痛點反轉**：「別再{痛點}——{你的解法}」——例：別再用 LINE 記帳，報價三分鐘出手。
3. **身分宣告**：「{地區}{最...的}{行業身分}」——例：台中最懂送禮的甜點工作室。

## 接案／個人品牌

- 標語方向：作品說話（「案子會說話——看完再決定要不要找我」）／專業×溫度（「把你的想法，做成能上線的樣子」）
- 亮點卡三維度：**專業能力**（會什麼）→ **合作方式**（怎麼一起工作：報價透明/準時交付）→ **信任證明**（做過什麼/幾年經驗）
- 關於骨架：我是誰＋為什麼做這行＋接什麼樣的案子（也講不接什麼，更顯專業）
- CTA 建議：看作品再說／預約諮詢

## 工作室／團隊

- 標語方向：手作感×堅持（「每一件都當自己的作品做」）／小而精（「小團隊，只做做得好的事」）
- 亮點卡三維度：**作品/服務特色** → **製作過程的堅持**（材料/工法/當天現做）→ **客製彈性**（能為你調什麼）
- 關於骨架：工作室的起點故事＋規模誠實講（小=用心的證明）＋服務範圍
- CTA 建議：加 LINE 聊聊／預約參觀

## 專業服務（顧問／設計／律師／會計）

- 標語方向：結果導向（「讓數字自己說服你的客戶」）／降低門檻（「第一次諮詢，先聽你說」）
- 亮點卡三維度：**專長領域**（明確列，不貪多）→ **服務流程**（第一次見面會發生什麼——去神秘化）→ **收費透明**（怎麼計費，敢寫就是信任）
- 關於骨架：資歷（簡潔有力，兩三行）＋服務理念＋適合找我的情況
- CTA 建議：預約諮詢（給時段感：「本週還有 X 個時段」型語感——但別造假數字）

## 店家／餐飲

- 標語方向：招牌品項打頭（「就為那一碗來的」）／情境召喚（「下班後的第一口放鬆」）
- 亮點卡三維度：**招牌品項**（一個就好，打深）→ **食材/現做的堅持** → **怎麼買/怎麼去**（外送/預訂/停車——實用資訊就是體貼）
- 關於骨架：開店的理由＋每天的節奏（幾點烤第一爐——畫面感）＋地址營業時間
- CTA 建議：加 LINE 訂購／看菜單／導航到店
